# 程序员薪资分析系统 python1192

## 项目概述

本项目是基于 Python 的程序员薪资分析系统，旨在为用户提供一个全面了解程序员薪资情况的平台。系统包括前台用户界面和后台管理系统，涵盖招聘信息、论坛交流、个人中心等多个功能模块。

## 技术栈

- **后端语言**: Python
- **框架**: Django
- **数据库**: Mysql
- **架构**: B/S

## 功能模块

### 前台功能

- 首页
- 招聘信息
- 论坛交流
- 公告信息
- 个人中心

### 管理员功能

- 首页
- 我的信息
- 用户管理
- 招聘信息管理
- 论坛交流
- 系统管理

## 系统角色

- **普通用户**: 访问前台功能，查看和发布招聘信息、参与论坛交流等。
- **管理员**: 管理后台，负责用户管理、招聘信息管理和系统设置等。
